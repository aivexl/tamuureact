# Blog SEO & Image Metadata Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement manual SEO controls (SEO Title, Description, Keywords) and Image Alt text in the Blog Admin Editor and ensure they are utilized on the public blog pages.

**Architecture:** Update the React-based Admin Editor (`apps/web`) to manage new SEO state variables and display corresponding UI fields in the sidebar. Update the Next.js blog page (`apps/main`) to prioritize these manual SEO values over automatic fallbacks.

**Tech Stack:** React (Vite), Next.js (App Router), Lucide Icons, Tailwind CSS, Cloudflare D1.
